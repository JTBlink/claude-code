# 前端设计插件

> 生成独特的、生产级前端界面，避免通用 AI 美学。

## 概述

Claude 在前端工作中自动使用此技能。创建生产就绪的代码，具有：

- 大胆的美学选择
- 独特的字体和调色板
- 高影响力的动画和视觉细节
- 上下文感知的实现

## 使用方式

```
"为音乐流媒体应用创建仪表板"
"为 AI 安全初创公司构建落地页"
"设计一个带暗黑模式的设置面板"
```

Claude 会选择清晰的美学方向，并实现具有细致入微的细节关注的生产级代码。

## 设计原则

### 大胆的美学选择

在编码前，理解上下文并致力于**大胆**的美学方向：

- **目的**: 这个界面解决什么问题？谁使用它？
- **基调**: 选择一个极端：极简主义、混沌最大主义、复古未来主义、有机/自然、奢华/精致、俏皮/玩具风格、编辑/杂志、野兽派/原始、装饰艺术/几何、柔和/粉彩、工业/实用主义等
- **约束**: 技术要求（框架、性能、可访问性）
- **差异化**: 什么让这个界面**难忘**？有什么东西让人记住？

**关键**: 选择清晰的概念方向并精确执行。大胆的最大主义和精致的极简主义都有效——关键是意图性，而非强度。

### 设计指南

**排版**: 选择美丽、独特、有趣的字体。避免 Arial 和 Inter 等通用字体；选择能提升前端美学的独特选择；意想不到、有特色的字体选择。将独特的展示字体与精致的正文字体配对。

**颜色和主题**: 致力于连贯的美学。使用 CSS 变量保持一致性。主导色配鲜明强调色优于胆怯、均匀分布的调色板。

**动效**: 使用动画进行效果和微交互。HTML 优先使用纯 CSS 解决方案。React 可用时使用 Motion 库。专注于高影响力时刻：一个精心编排的页面加载配交错显示（animation-delay）比分散的微交互带来更多愉悦。使用滚动触发和惊喜的悬停状态。

**空间构图**: 意想不到的布局。不对称。重叠。对角线流动。打破网格的元素。慷慨的负空间或受控的密度。

**背景和视觉细节**: 创造氛围和深度而非默认纯色。添加匹配整体美学的上下文效果和纹理。应用创意形式如渐变网格、噪点纹理、几何图案、分层透明度、戏剧性阴影、装饰性边框、自定义光标和颗粒叠加。

## 避免的内容

**切勿**使用通用 AI 生成的美学：

- 过度使用的字体家族（Inter、Roboto、Arial、系统字体）
- 陈词滥调的配色方案（特别是白色背景上的紫色渐变）
- 可预测的布局和组件模式
- 缺乏上下文特定特征的千篇一律设计

**创意解读**，做出感觉真正为上下文设计的意想不到的选择。没有两个设计应该相同。在明暗主题、不同字体、不同美学之间变化。**切勿**在生成中收敛于常见选择。

**重要**: 匹配实现复杂度与美学愿景。最大主义设计需要精心制作的代码和大量动画效果。极简主义或精致设计需要克制、精确，以及对间距、排版和微妙细节的仔细关注。优雅来自于良好地执行愿景。

```
创建一个音乐流媒体应用的仪表板
```

### Claude 会
1. 分析上下文和目的
2. 选择独特的美学方向（如：暗黑、霓虹、赛博朋克风格）
3. 选择独特的字体组合
4. 创建有影响力的动画和交互
5. 实现生产级代码
