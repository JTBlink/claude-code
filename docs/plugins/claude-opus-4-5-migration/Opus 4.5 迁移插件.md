# Claude Opus 4.5 迁移插件

> 将代码和提示词从 Sonnet 4.x 和 Opus 4.1 迁移到 Opus 4.5。

## 概述

此技能更新你的代码和提示词以兼容 Opus 4.5。它自动化迁移过程，处理模型字符串、beta headers 和其他配置细节。如果在迁移后遇到任何 Opus 4.5 相关问题，可以继续使用此技能调整提示词。

## 使用方式

```
"将我的代码库迁移到 Opus 4.5"
```

## 迁移内容

- **模型字符串替换**: 更新模型标识符
- **Beta headers 更新**: 调整 API 请求头
- **提示词调整**: 优化提示词以适应新模型特性

## 最佳实践

参考我们的 [提示词指南](https://platform.claude.com/docs/en/build-with-claude/prompt-engineering/claude-4-best-practices) 了解 Claude 模型提示的最佳实践。
