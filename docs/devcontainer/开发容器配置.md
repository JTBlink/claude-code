## 快速开始

### VS Code 中使用

1. **安装 Dev Containers 扩展**
   - 打开 VS Code
   - 扩展商店搜索 "Dev Containers"
   - 安装 Microsoft 官方扩展

2. **在容器中打开**
   - 命令面板 (`Ctrl+Shift+P` / `Cmd+Shift+P`)
   - 选择 `Dev Containers: Reopen in Container`
   - 等待容器构建和启动

3. **开始开发**
   - 容器启动后，VS Code 会自动连接到容器
   - 所有开发都在隔离的容器环境中进行

### 命令行使用

```bash
# 构建容器
docker-compose build

# 启动容器
docker-compose up -d

# 进入容器
docker-compose exec claude-code bash
```

## 配置说明

### devcontainer.json

**位置**: `.devcontainer/devcontainer.json`

**主要配置项**:

```json
{
  "name": "Claude Code Development",
  "dockerComposeFile": "docker-compose.yml",
  "service": "claude-code",
  "workspaceFolder": "/workspace",
  
  // 功能配置
  "features": {
    "ghcr.io/devcontainers/features/node": {
      "version": "18"
    },
    "ghcr.io/devcontainers/features/git": {}
  },
  
  // 转发端口
  "forwardPorts": [3000],
  
  // 启动后执行
  "postCreateCommand": "npm install",
  
  // 扩展推荐
  "extensions": [
    "dbaeumer.vscode-eslint",
    "esbenp.prettier-vscode"
  ]
}
```

### Dockerfile

**位置**: `.devcontainer/Dockerfile`

**内容说明**:

```dockerfile
# 基础镜像
FROM mcr.microsoft.com/devcontainers/typescript-node:18

# 安装依赖工具
RUN apt-get update && export DEBIAN_FRONTEND=noninteractive \
    && apt-get -y install --no-install-recommends \
    git \
    gh \
    ripgrep

# 安装 Claude Code
RUN npm install -g @anthropic-ai/claude-code

# 设置工作目录
WORKDIR /workspace
```

### docker-compose.yml

**位置**: `.devcontainer/docker-compose.yml`

**内容说明**:

```yaml
version: '3.8'
services:
  claude-code:
    build:
      context: .
      dockerfile: Dockerfile
    volumes:
      - ..:/workspace:cached
    command: sleep infinity
    network_mode: service:
```

### init-firewall.sh

**位置**: `.devcontainer/init-firewall.sh`

**功能**: 初始化容器内防火墙规则

**使用场景**: 限制容器网络访问，增强安全性

## 使用指南

### 环境变量配置

在项目根目录创建 `.env` 文件：

```bash
# Anthropic API Key
ANTHROPIC_API_KEY=sk-ant-xxxxx

# GitHub Token（可选）
GITHUB_TOKEN=ghp_xxxxx

# 其他配置
NODE_ENV=development
```

### 端口转发

容器内服务可通过端口转发访问：

```json
{
  "forwardPorts": [
    3000,  // Web 服务
    5432   // 数据库
  ]
}
```

### 数据持久化

使用卷挂载持久化数据：

```yaml
volumes:
  - ..:/workspace:cached      # 代码目录
  - node_modules:/workspace/node_modules  # 依赖

volumes:
  node_modules:
```

### 扩展配置

容器内自动安装的 VS Code 扩展：

| 扩展 | 说明 |
|------|------|
| `vscode-eslint` | ESLint 代码检查 |
| `prettier-vscode` | Prettier 格式化 |
| `gitlens` | Git 增强功能 |

## 容器内开发

### 运行 Claude Code

```bash
# 在容器内运行
claude

# 或使用 Docker 命令
docker-compose exec claude-code claude
```

### 安装依赖

```bash
# 容器内安装
npm install

# 或从主机运行
docker-compose exec claude-code npm install
```

### 运行测试

```bash
# 容器内运行测试
npm test

# 或特定测试
docker-compose exec claude-code npm run test:unit
```

### 调试配置

`.vscode/launch.json`:

```json
{
  "version": "0.2.0",
  "configurations": [
    {
      "type": "node",
      "request": "launch",
      "name": "Debug in Container",
      "skipFiles": ["<node_internals>/**"],
      "runtimeExecutable": "docker-compose",
      "runtimeArgs": [
        "exec",
        "claude-code",
        "node",
        "--inspect-brk",
        "/workspace/src/index.js"
      ]
    }
  ]
}
```

## 故障排查

### 问题：容器构建失败

**症状**:
```
Error: Cannot find base config file
```

**解决方案**:
```bash
# 清理缓存
docker-compose down
docker system prune -a

# 重新构建
docker-compose build --no-cache
```

### 问题：无法连接容器

**症状**: VS Code 无法连接到容器

**解决方案**:
```bash
# 检查容器状态
docker-compose ps

# 查看容器日志
docker-compose logs claude-code

# 重启容器
docker-compose restart
```

### 问题：端口无法访问

**症状**: 转发的端口无法访问

**解决方案**:
```bash
# 检查端口占用
docker-compose ps

# 修改转发端口
# 编辑 .devcontainer/devcontainer.json
{
  "forwardPorts": [3001]  // 改为其他端口
}
```

### 问题：性能缓慢

**症状**: 容器内操作响应慢

**解决方案**:
```bash
# 增加容器资源限制
# 编辑 docker-compose.yml
services:
  claude-code:
    deploy:
      resources:
        limits:
          cpus: '2'
          memory: 4G
```

## 最佳实践

### 容器大小优化

```dockerfile
# 使用多阶段构建
FROM node:18 AS builder
WORKDIR /workspace
COPY package*.json ./
RUN npm ci --only=production

FROM node:18-alpine
COPY --from=builder /workspace/node_modules ./node_modules
```

### 开发体验优化

```json
{
  // 使用缓存卷
  "mounts": [
    "source=${localEnv:HOME}/.npm,target=/home/node/.npm,type=bind"
  ],
  
  // 环境变量继承
  "containerEnv": {
    "NODE_ENV": "development"
  }
}
```

### 安全配置

```json
{
  // 限制容器权限
  "runArgs": [
    "--cap-drop=ALL",
    "--cap-add=NET_BIND_SERVICE"
  ],
  
  // 只读文件系统（可选）
  "containerUser": "node"
}
```
