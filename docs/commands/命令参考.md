## 快速速查表

### 常用内置命令
| 命令 | 描述 |
|------|------|
| `/help` | 显示帮助信息 |
| `/btw` | 附带问题提问（不污染主对话上下文） |
| `/editor` | 打开多行编辑器模式 |
| `/compact` | 压缩会话历史，节省 token |
| `/clear` | 清除当前会话历史 |

### 常用自定义/插件命令
| 命令 | 描述 |
|------|------|
| `/commit-push-pr` | 一键完成代码提交、推送到远程、创建 Pull Request |
| `/feature-dev` | 启动 7 阶段结构化功能开发工作流 |
| `/code-review` | 自动化 PR 代码审查 |
| `/ralph-loop` | 启动自主迭代循环 |
